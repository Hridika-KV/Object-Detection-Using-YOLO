# YOLO Object Detection

## Description
This project uses the YOLO model (Ultralytics) to detect objects in images and videos.  
It draws bounding boxes around detected objects and saves the output.

---

## Features
- Detects objects in images
- Detects objects in videos
- Single Python file for both image and video
- Saves output results automatically
- Simple input selection

---

## Technologies Used
- Python
- YOLO (Ultralytics)
- OpenCV
- PyTorch
- NumPy

---

## Project Structure
ObjectDetection/
├── objectdetection.py
├── sample_inputs/   (2 images + 1 video)
├── run/             (output results)
└── README.md

---

## How It Works
1. Run the Python file  
2. Select image or video input  
3. YOLO detects objects  
4. Output is saved in the `run/` folder
